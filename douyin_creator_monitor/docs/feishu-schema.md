# Feishu Schema Notes

## 达人基础信息表

真实表 ID 不写入可提交文档。请在本地私有配置中维护：

```text
douyin_creator_monitor/local/feishu-ids.md
```

关键字段：

- `达人昵称`
- `达人主页地址`
- `所属平台`
- `抖音UID`
- `SecUID`
- `账号ID`
- `账号简介`
- `粉丝数`
- `关注数`
- `获赞数`
- `作品数`
- `IP属地`
- `所在地区`
- `最近检查时间`
- `最近发稿时间`
- `主页采集状态`
- `账号状态`
- `主页原始数据`
- `作品表名称`
- `作品表ID`
- `作品表链接`

字段语义：

- `创建时间`: 飞书记录创建时间，不是抖音账号创建时间。
- `最近发稿时间`: 该达人最新非置顶作品的 `create_time` 转北京时间；测试时直接使用主页列表中最新作品时间。
- `所属平台`: 当前按主页 URL 域名推断，`douyin.com` 写为 `抖音`。

## 作品表规则

- 每个作品表只记录一个达人。
- 作品表名称使用达人昵称。
- 达人基础信息表通过 `作品表名称`、`作品表ID`、`作品表链接` 关联到对应作品表。

作品表关键字段：

- `抖音作品ID`
- `作品链接`
- `作品类型`
- `是否置顶`
- `作品标题`
- `原始文案`
- `话题标签`
- `封面图URL`
- `页面可见数字`
- `页面可见数字含义`
- `发布时间`
- `详情页状态`
- `视频时长秒`
- `点赞数`
- `评论数`
- `收藏数`
- `分享数`
- `通义听悟状态`
- `转写状态`
- `转写完成时间`
- `转写文案`
- `词库纠错文案`
- `转写原始结果`
- `转写纠错报告`
- `ima状态`
- `百度网盘状态`
- `本地知识库状态`
- `采集状态`
- `最近采集时间`
- `记录时间`
- `原始作品数据`

语音转写字段建议：

- `转写文案`: 火山引擎 ASR 返回并抽取出的完整文案。
- `词库纠错文案`: 经过 `domain_glossary.json` 规则修正后的文案，后续优先用于飞书摘要、ima 和本地知识库。
- `转写状态`: 建议值包括 `待转写`、`已完成`、`失败`。
- `转写完成时间`: 脚本写回时的本地完成时间。
- `转写原始结果`: 可选，保存 ASR 原始 JSON，便于排错；如果表格过重，可以不建或不写这个字段。
- `转写纠错报告`: 可选，保存命中的替换规则和候选新词报告。

写回规则：

- 每个作品表只记录一个达人，写回时必须明确目标作品表 ID。
- 脚本默认用 `抖音作品ID` 搜索对应记录，然后更新同一条记录。
- 不要用标题、发布时间或作品链接做唯一定位，避免误写到相似作品。
